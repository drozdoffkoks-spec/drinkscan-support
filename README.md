# DrinkScan Support Site

Static support site for DrinkScan.

Target repository:

- Owner: `drozdoffkoks-spec`
- Repository: `drinkscan-support`
- GitHub Pages URL: `https://drozdoffkoks-spec.github.io/drinkscan-support/`
- Google Play account deletion URL: `https://drozdoffkoks-spec.github.io/drinkscan-support/delete-account/`

## Files

- `index.html` - support home
- `delete-account/index.html` - account and data deletion instructions
- `privacy/index.html` - privacy summary
- `terms/index.html` - terms summary

## Enable GitHub Pages

1. Create the GitHub repository `drozdoffkoks-spec/drinkscan-support`.
2. Push these files to the repository's default branch.
3. Open the repository on GitHub.
4. Go to Settings -> Pages.
5. Under "Build and deployment", set Source to "Deploy from a branch".
6. Select the default branch and `/ (root)`.
7. Save.
8. Wait for GitHub Pages to publish the site.

Use this URL in Google Play Console for account deletion:

`https://drozdoffkoks-spec.github.io/drinkscan-support/delete-account/`
